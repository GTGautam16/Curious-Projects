# 🧠 Curious Projects

A collection of curiosity-driven engineering experiments built to understand how systems work under the hood.

This repository focuses on learning through building, debugging, breaking, rebuilding, and deeply understanding the logic behind software systems and engineering concepts.

Instead of only using high-level tools and frameworks, the purpose here is to explore the internal mechanics of technologies through hands-on experimentation and practical implementation.

---

## 🚀 Purpose

* Learn concepts by building from scratch
* Explore how software behaves internally
* Improve system thinking and debugging skills
* Practice clean architecture and problem-solving
* Develop a deeper engineering mindset through experimentation

---

## 🛠️ Areas of Interest

* Backend Engineering
* Networking Fundamentals
* System Design Concepts
* Python Engineering
* Automation
* Web Technologies
* Design Patterns
* Low-Level Software Concepts

---

## 📖 Philosophy

Curiosity is one of the strongest engineering tools.

This repository exists as a space to:

* explore ideas,
* experiment freely,
* understand systems deeply,
* and document the learning journey along the way.
